# RT1 Smart Attendance
ระบบบันทึก ตรวจสอบ และสรุปรายงานการมาเรียนออนไลน์  
สำหรับโรงเรียนรวมไทยพัฒนา 1

## Features
- บันทึกการมาเรียนรายห้อง
- ตรวจสอบห้องที่ยังไม่บันทึก
- ตั้งค่าเทอมและวันหยุด
- สรุปรายงานรายวัน
- ส่งออก PDF
- ใช้งานผ่าน Google Apps Script และ Google Sheets

## Files
- `index.html` หน้าแบบฟอร์มบันทึกข้อมูล
- `check_missing.html` หน้าตรวจสอบการบันทึก
- `admin.html` หน้าตั้งค่าระบบ
- `search.html` หน้าสรุปรายงาน
- `code.gs` ฟังก์ชันหลักฝั่ง Google Apps Script

## Google Sheets Structure
### สถิตินักเรียนทั้งหมด
ระดับชั้น | ชาย | หญิง | รวม | ครูประจำชั้น

### บันทึกการมาเรียน
วันที่ | ระดับชั้น | ชายทั้งหมด | หญิงทั้งหมด | รวมทั้งหมด | ชายมา | หญิงมา | รวมมา

### ตั้งค่าเทอม
ปีการศึกษา | ภาคเรียน | วันที่เปิดเทอม | วันที่ปิดเทอม | สถานะ

### วันหยุด
วันที่เริ่ม | วันที่สิ้นสุด | ประเภท | รายการ | หมายเหตุ | สถานะ
